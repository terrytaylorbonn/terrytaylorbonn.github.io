#### **[2.3.0 TF concepts](/2.3.0-tf-concepts/)** (26.0616)

An LLM consists of:
- Transformer (TF) (the NN)
- Internal agent (iAgent) (interfaces with TF (via tokens) and the agent scripts)

See also __ziptieai_BOOK2_LLM_.docx the [Gdrive](https://drive.google.com/drive/folders/1-Adawag9uA8_bq-hDF-nOuPYaRLz1eEO).
